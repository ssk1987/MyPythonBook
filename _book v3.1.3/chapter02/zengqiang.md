## 增强运算符

| 运算符 | 例子 | 等价 |
| :--: | :--: | :--: |
| += | y += x | 等同于 y = y + x |
| -= | y -= x | 等同于 y = y - x |
| *= | y *= x | 等同于 y = y * x |
| /= | y /= x | 等同于 y = y / x |
| //= | y //= x	| 等同于 y = y // x |
| %= | y %= x | 等同于 y = y % x |
| \*\*= | y \*\*= x | 等同于 y = y \** x |

> 注意：“ += ” 中间不能加空格！

**【操作】源代码：**

```
>>> a = 2
>>> a += 2
>>> a
4

>>> a -= 2
>>> a
2

>>> a *= 2
>>> a
4

>>> a /= 2
>>> a
2.0

>>> b = 5
>>> b //= 2
>>> b
2

>>> b **= 2
>>> b
4

>>> b %= 2
>>> b
0

```
