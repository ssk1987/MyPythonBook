
## 单引和双引号的区别


1.单引号内的双引号不算结束符

2.双引号内的单引号不算结束符

```
演示代码：

a = '"我们一起学Python"'

b = "'我们一起学Python'"

print(a)
print(b)
```
运行结果：
```
"我们一起学Python"

'我们一起学Python'
```

## 三引号作用

1.换行会自动转换为换行符\n

2.三引号内可以包含单引号和双引号

```
演示代码：

c = """
----------------
我们一起学Python
共同进步
----------------
"""

print(c)

# -----------------------------

d = """我们一起来学Python，"共同进步"，'努力前进'"""

print(d)
```


运行结果：

```
----------------
我们一起学Python
共同进步
----------------

# -----------------------------

我们一起来学Python，"共同进步"，'努力前进'
```
