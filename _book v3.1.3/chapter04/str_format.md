
## 字符串格式化


1.定义：

生成一定格式的字符串。

2.语法：
```
字符串%(变量)

"我的名字是%s,年龄是%s" % (name, age)
```
3.类型码：

| 类型 | 说明 |
| -- | -- |
| %s | 字符串 |
| %d | 整数 |
| %f | 浮点数 |

-----

演示：

```
# 字符串格式化
a = "1"
b = "2"
# "请输入" + str(a) + "+" + str(b) + "=?"

# ----------------------------------------------
# 在字符串中插入变量
# 请输入1+2=?
# 字符串拼接（缺点：乱）
str01 = "请输入" + a + "+" + b + "=?"

str02 = "请输入%s+%s=?" % (a, b)

str03 = "请输入%s+%.1f=?" % ("1", 10.5678)
print(str03)
```
